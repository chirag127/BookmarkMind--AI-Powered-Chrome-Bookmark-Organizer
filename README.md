# BookmarkMind - AI Bookmark Organizer

AI-powered Chrome extension that automatically categorizes and organizes your bookmarks using Google Gemini, Cerebras, and Groq AI models.

## Features

-   **16 AI Models**: Gemini (7), Cerebras (4), Groq (5) with intelligent fallback
-   **FMHY-Style Functional Organization**: Categories by function, not provider
-   **Learning System**: Learns from user corrections only
-   **Snapshot/Undo**: 10 snapshots with full restore capability
-   **Model Comparison Dashboard**: Performance tracking, A/B testing, cost alerts
-   **Live Title Fetching**: Automatically fetches page titles for better categorization context
-   **Enhanced Security**: Robust API key validation and secure storage
-   **Folder Health Scores**: 0-100 scoring with smart suggestions

See [AGENTS.md](AGENTS.md) for development commands and technical details.
