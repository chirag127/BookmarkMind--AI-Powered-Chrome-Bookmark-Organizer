Powerful AI Features

Smart AI Categorization

Advanced Gemini AI analyzes your bookmarks and creates intelligent hierarchical categories based on content, domain, and context.

Automatic Title Enhancement

AI generates descriptive, informative titles for your bookmarks, making them easier to identify and search.

Duplicate URL Removal

Automatically detects and removes duplicate bookmarks that point to the same webpage with different URL fragments.

Hierarchical Organization

Creates deep, meaningful folder structures with up to 7 levels of categorization for complex bookmark collections.

Batch Processing

Process up to 50 bookmarks at once with intelligent rate limiting to avoid API restrictions and ensure reliable operation.

Privacy-First Design

Your bookmarks are processed securely with optional AgentRouter fallback. No personal data is stored or shared.

Easy Installation

1

Download Extension

Download the latest release from GitHub or clone the repository to your local machine.

2

Enable Developer Mode

Go to chrome://extensions/ and toggle "Developer mode" in the top right corner.

3

Load Extension

Click "Load unpacked" and select the BookmarkMind extension folder.

4

Configure API Key

Get a free Gemini API key from Google AI Studio and configure it in the extension settings.

5

Start Organizing

Click the extension icon and use "Sort Bookmarks Now" to begin AI-powered organization.
